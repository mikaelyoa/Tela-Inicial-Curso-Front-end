HTML

HTML (HyperText Markup Language) é a linguagem padrão utilizada para criar páginas da web. Ele é um tipo de linguagem de marcação, o que significa que usa uma série de "marcas" ou "tags" para definir o conteúdo e a estrutura de um documento. Em essência, o HTML descreve o que deve aparecer em uma página da web, como textos, imagens, links, tabelas, vídeos e outros elementos interativos.

CSS

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar páginas HTML, definindo como os elementos devem ser exibidos no navegador. Enquanto o HTML estrutura o conteúdo, o CSS controla a aparência visual, como cores, fontes, margens, espaçamentos e layouts. Em resumo, o CSS é crucial para dar vida e estética às páginas web, proporcionando uma interface visual mais agradável e profissional.

JavaScript

JavaScript é uma linguagem de programação amplamente utilizada para tornar as páginas web interativas e dinâmicas. Diferente do HTML, que estrutura o conteúdo, e do CSS, que estiliza a aparência, o JavaScript (JS) permite adicionar funcionalidades como animações, validação de formulários, controle de eventos e comunicação com servidores. Ele é usado tanto no front-end quanto no back-end (com o Node.js), fazendo dele uma linguagem extremamente versátil e poderosa.
